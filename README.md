# AutoMaster — Professional Car Service Website

A modern, high-performance, and fully responsive multi-page website designed for a professional car service and tire repair business. This project features a clean user interface, seamless multi-page navigation, and a focus on conversion-driven elements.

## 🔗 Live Demo
Explore the fully operational live website here:
👉 **[https://qq-astral666.github.io/tire-servise/](https://qq-astral666.github.io/tire-servise/)**

---

## 🛠️ Tech Stack & Methods
* **Markup:** Semantic HTML5 for structural integrity, better SEO ranking, and accessibility.
* **Styling:** Custom CSS3 with optimized component structures and smooth interactive states.
* **Logic:** Vanilla JavaScript for interface controls, slideshows, and dynamic user interactions.
* **Responsiveness:** Fully optimized layout adapting flawlessly to desktops, tablets, and mobile smartphones.

---

## 🚀 Key Features Implemented

1. **Multi-Page Architecture:** Features dedicated semantic pages to distribute content logically and look like an enterprise-level business site.
2. **Interactive Showcase Sliders:** Custom JavaScript-driven slider elements embedded into the layout to switch smoothly between content previews.
3. **Structured Service Catalog:** Comprehensive price lists and service cards designed for quick readability and high conversion.
4. **Conversion-Oriented Booking:** Strategic Call-to-Action (CTA) sections with optimized input forms for effortless booking and appointment scheduling.
5. **Clean Codebase:** Adheres to modern web standards, featuring well-organized stylesheets and efficient script handles.

---

## 📂 Project Structure & Pages

### 📄 HTML Pages
* **[Homepage (`index.html`)](https://github.io)** — Hero banner, main business introduction, core features, brand trust blocks, and scheduling forms.
* **[Services Page (`services.html`)](https://github.ioservices.html)** — Full list of tire fitting, wheel balancing, and professional diagnostic procedures with integrated price tags.
* **[Contacts Page (`contacts.html`)](https://github.iocontacts.html)** — Interactive company placement overview, working hours, active phone hotlines, and message boxes.

### 🎨 Styles & Logic Assets
* `style.css` — Central stylesheet featuring responsive design modules, custom layout overrides, global variables, and hover-state animations.
* `script.js` — Core application script management containing active slide-switching algorithms (`moveSlide`), touch/click event bindings, and interactive menu routines.
* `img/` — Optimized graphics folder storage housing internal asset files, background masks, and service preview photos.
